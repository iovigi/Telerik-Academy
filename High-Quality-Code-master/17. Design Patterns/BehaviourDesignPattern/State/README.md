#State
State е поведенчески шаблон за дизайн, използва се за динамична смяна на състояния на даден обект, чрез които се определя поведението на обекта. Чрез State се изгражда state machine. Логиката на състоянието е затворена в самото състояние. Всяко състояне знае за друго състояние
